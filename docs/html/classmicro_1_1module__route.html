Detailed Description

module_route allows users of micro++ to group URL strings, callback functions, and method lists together, so that they may be bound to the application at a later time.

Users can declare a module, which is typdef'd as a vector of module_route**s. Then, the user can fill the **module and register it with an instance of app. All module_route**s within that **module will be converted to url endpoints.
