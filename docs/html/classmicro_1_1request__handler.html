void micro::request_handler::add_route ( url_route  route)

Adds a callback url route to the list of routes this application can service.

Routes are matched in the order in which they were installed. If there is a collision between a URL route and a file within the static folder, the file in the static folder will always be matched to the URL first.

Parameters
routethe url_route object representing the route to be added.
